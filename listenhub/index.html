nexu 是一个可以一键安装的开源 OpenClaw 桌面客户端，让你在本地用 AI 操控一切。

GitHub → github.com/nexu-io/nexu，觉得有用的话帮忙点个 Star 支持一下 🌟

在 nexu 里对 agent 说一句话，它就能帮你生成一期完整的播客节目。不是那种机械朗读的 TTS，而是两个人有来有回、有观点有碰撞的深度对话。

不只是播客——PPT、解说视频，在 nexu 里都是一句话的事。 你说需求，agent 帮你搞定从安装工具到交付成品的全流程。

谁会用到？

📝 自媒体博主 — 一个选题同时产出文章、播客、短视频，一鱼三吃

🎓 培训讲师 / 教育工作者 — 把课件和文档变成带配音的教学视频，5 分钟出片

🚀 产品经理 / 创业者 — 快速生成产品介绍视频和 pitch deck 配音，不用求设计师

💻 技术博主 / 开发者 — 技术文章一键转播客，让内容触达不看文字的受众

📊 知识工作者 — 行业报告、竞品分析秒变解说 PPT，发给团队直接看

下面用一个真实案例跑一遍完整流程。

今天我用 nexu agent 完成了一套完整的内容生成流程：

输入：一句话描述主题——"Harness Engineering：AI驱动DevOps自动化"

输出：

🎙️ 一期 12 分钟的双人深度播客

📊 一份 5 页的解说 PPT（带 AI 配图 + 配音）

🎬 一个完整的解说视频

全程零手动操作，从输入到产出大概 5 分钟。

核心工具是 ListenHub CLI —— 一个可以在终端运行的内容生成工具，支持播客、TTS、解说视频、PPT、AI 图片等多种格式。

把它装到 nexu 里之后，agent 就多了一项能力：把任何主题变成多媒体内容。

第一步：安装 ListenHub CLI

把这个链接发给你的 nexu agent：

https://github.com/marswaveai/listenhub-cli
安装这个 CLI

agent 会自动帮你下载、安装、配置好一切。装完后它还会问你要不要登录——点确认就行。

第二步：一句话生成播客

对 nexu agent 说一句话：

"帮我生成一个关于 Harness Engineering 的深度播客"

agent 自动选择 deep 模式，先做深度调研，再生成对话脚本，最后合成音频。

5 分钟后拿到了一期 12 分钟的深度播客。两位主播（原野 & 高晴）从 DevOps 的痛点聊到 AI Agent 的操作系统，从云成本优化聊到工程师角色的转变，甚至还讨论了"信任债"这种前沿话题。

第三步：生成解说 PPT

继续对 agent 说："帮我把这个内容做成解说 PPT"。agent 调用 ListenHub 的 explainer 接口，生成一份带 AI 配图和配音的解说幻灯片：

生成的 5 页 PPT 覆盖了：

1. AI 奇点时刻 — 从工匠到系统导演

2. 代码洪流 — 传统 DevOps 的失控危机

3. 智能净化 — CI/CD 测试缩短 80% + 自动回滚

4. 云端降本 — AutoStopping 省 70%

5. 信任债 — Harness 作为 AI 的"精密马具"

第四步：一键导出视频

PPT 生成后，agent 自动调用视频导出接口。15 秒后，一个完整的解说视频就出来了——slides 配上旁白配音，可以直接发到任何平台。

你可能会问：这些功能 ListenHub 网站上不也有吗？为什么要在 nexu 里用 CLI？

因为 agent 能帮你串起整个工作流。

我实际的使用过程就是一句话：

"帮我生成一个关于 Harness Engineering 的解说播客"

一句话。agent 自动完成了：安装 CLI → 登录授权 → 选择参数 → 生成播客 → 生成 PPT → 导出视频 → 部署预览页 → 把结果发给我。

我不需要记任何命令、任何参数、任何 API 文档。agent 都帮我处理了。

这就是 nexu 的理念：AI 站在你旁边，帮你操控一切工具。 你只需要说你想要什么。

1. 下载 Nexu（可一键安装的开源 OpenClaw 桌面客户端）：github.com/nexu-io/nexu

2. 把 ListenHub CLI 链接 发给你的 agent，说"帮我装这个"

3. agent 装完后会帮你登录，按提示确认即可

4. 对 agent 说："帮我生成一个关于 XXX 的播客"

就这么简单。
